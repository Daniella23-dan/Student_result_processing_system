A simple, memory_based console application built in Rust to manage student records, scores, and grade calculations.
-** Add Students**: Create records with unique IDs, names, and ages.
-**Score Management**: Add multiple exam scores to existing students.
-**Reporting**
-View a summary table of all students.
-View a detailed report for an individual student(Total,Average, and Grade).
-**Automated Grading**: Calculates gardes (Execellent, Very Good, Good, and Fail.) based on average scores.
__-**Removal**: Delete studeny records by ID.
